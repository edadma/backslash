distinct
========

### Description

Removes duplicates from a sequence.

### Input

Takes one sequence argument.

### Output

Returns a new sequence containing only distinct values.

### Example

    \seq {1 2 3 2 4 3 5} | distinct

output

    [1, 2, 3, 4, 5]

### Exceptions

If the argument is not a sequence, an exception is thrown.
